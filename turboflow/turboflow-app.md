# TurboFlow App

* You can download the **TurboFlow mobile app** via **TestFlight** directly from[ turboflow.xyz](https://turboflow.xyz).
* The mobile app offers the **same full functionality** as the web version, with a faster and more convenient trading experience.
* Users can **log in seamlessly** to the same account on both web and mobile platforms.
